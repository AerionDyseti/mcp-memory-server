# When Stuck Skill

You apply proven strategies for getting unstuck when facing challenging problems or blockers.

## Strategies for Getting Unstuck

### 1. Take a Break
- Step away for 5-15 minutes
- Let your subconscious work on it
- Return with fresh perspective

### 2. Rubber Duck Debugging
- Explain the problem out loud
- Walk through code line by line
- Often reveals the issue

### 3. Start from Scratch
- Create minimal reproduction
- Build up from working state
- Isolate the problem

### 4. Change Your Approach
- Try a different solution
- Use different tools
- Research alternative methods

### 5. Seek Help
- Ask a colleague
- Search Stack Overflow
- Review documentation
- Ask in community forums

### 6. Break It Down
- Divide into smaller problems
- Solve one piece at a time
- Build up to full solution

### 7. Work Backward
- Start from desired end state
- Trace backward to current state
- Identify missing steps

### 8. Check Assumptions
- List all assumptions
- Verify each one
- Often the issue is a false assumption

## When Invoked

1. **Recognize**: Admit you're stuck
2. **Document**: Write down what you've tried
3. **Choose Strategy**: Pick appropriate approach
4. **Apply**: Follow strategy systematically
5. **Reflect**: Learn what worked
6. **Prevent**: Avoid similar blocks
